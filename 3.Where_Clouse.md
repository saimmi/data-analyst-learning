## 🔹 WHERE Clause (Filtering Data)

### 📌 What is WHERE?

* Used to **filter records**
* Returns only rows that **match a given condition**

---

### 1️⃣ Basic Filtering

```sql id="w1a2b3"
SELECT *
FROM employee_salary;
```

👉 Returns all records (no filtering)

```sql id="w4c5d6"
SELECT *
FROM employee_salary
WHERE first_name = 'Leslie';
```

👉 Returns only rows where `first_name` is Leslie

---

### 2️⃣ Comparison Operators

```sql id="w7e8f9"
SELECT *
FROM employee_salary
WHERE salary > 50000;
```

👉 Returns employees with salary **greater than 50000**
👉 Tom and Jerry are excluded because their salary = 50000

```sql id="w10g11"
SELECT *
FROM employee_salary
WHERE salary >= 50000;
```

👉 Includes employees with salary **equal to 50000**

```sql id="w12h13"
SELECT *
FROM employee_salary
WHERE salary < 50000;
```

👉 Returns employees earning less than 50000

---

### 3️⃣ Filtering Using employee_demographics

```sql id="w14i15"
SELECT *
FROM employee_demographics;
```

👉 View all demographic data

---

### 4️⃣ Filter by Gender

```sql id="w16j17"
-- All female employees
SELECT *
FROM employee_demographics
WHERE gender = 'Female';
```

```sql id="w18k19"
-- All male employees
SELECT *
FROM employee_demographics
WHERE gender = 'Male';
```

```sql id="w20l21"
-- Employees who are NOT female
SELECT *
FROM employee_demographics
WHERE gender != 'Female';
```

---

### 5️⃣ Filter by Date

```sql id="w22m23"
-- Employees born after 1 Jan 1985
SELECT *
FROM employee_demographics
WHERE birth_date > '1985-01-01';
```

👉 Date format must be `'YYYY-MM-DD'`

---

### 6️⃣ Logical Operators (AND, OR, NOT)

```sql id="w24n25"
-- Employees who are male OR born after 1985
SELECT *
FROM employee_demographics
WHERE birth_date > '1985-01-01'
   OR gender = 'Male';
```

```sql id="w26o27"
-- Employees born after 1985 OR NOT male (i.e., female)
SELECT *
FROM employee_demographics
WHERE birth_date > '1985-01-01'
   OR NOT gender = 'Male';
```

```sql id="w28p29"
-- Leslie (age 44) OR anyone older than 55
SELECT *
FROM employee_demographics
WHERE (first_name = 'Leslie' AND age = 44)
   OR age > 55;
```

👉 Parentheses `()` control condition priority

---

### 7️⃣ LIKE Operator (Pattern Matching)

```sql id="w30q31"
-- Exact match
SELECT *
FROM employee_demographics
WHERE first_name = 'Jerry';
```

👉 Must match full value exactly

---

### 🔍 Using LIKE

```sql id="w32r33"
-- Names starting with 'Jer'
SELECT *
FROM employee_demographics
WHERE first_name LIKE 'Jer%';
```

```sql id="w34s35"
-- Names containing 'er'
SELECT *
FROM employee_demographics
WHERE first_name LIKE '%er%';
```

```sql id="w36t37"
-- Names starting with 'a'
SELECT *
FROM employee_demographics
WHERE first_name LIKE 'a%';
```

```sql id="w38u39"
-- Names ending with 'a'
SELECT *
FROM employee_demographics
WHERE first_name LIKE '%a';
```

---

### 🔤 Wildcards

* `%` → any number of characters
* `_` → exactly one character

```sql id="w40v41"
-- Names with exactly 3 letters starting with 'a'
SELECT *
FROM employee_demographics
WHERE first_name LIKE 'a__';
```

```sql id="w42x43"
-- Names starting with 'a' and at least 3 characters
SELECT *
FROM employee_demographics
WHERE first_name LIKE 'a__%';
```

---

### 8️⃣ LIKE with Dates

```sql id="w44y45"
-- Employees born in 1989
SELECT *
FROM employee_demographics
WHERE birth_date LIKE '1989%';
```

👉 Matches all dates starting with 1989
