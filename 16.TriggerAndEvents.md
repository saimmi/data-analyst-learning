## 🔹 Trigger (Automatic Action on INSERT)

### 📌 Create Trigger

```sql id="te2"
DELIMITER $$

-- Trigger to insert data into employee_demographics 
-- whenever a new row is added to employee_salary
CREATE TRIGGER employee_insert
AFTER INSERT ON employee_salary
FOR EACH ROW
BEGIN
    INSERT INTO employee_demographics (employee_id, first_name, last_name)
    VALUES (NEW.employee_id, NEW.first_name, NEW.last_name);
END $$

DELIMITER ;
```

## 🔹 Events (Scheduled Tasks)

### 📌 View Table

```sql id="te7"
SELECT *
FROM employee_demographics;
```

---

### 📌 Create Event

```sql id="te8"
DELIMITER $$

-- Event to delete employees whose age is 60 or above
CREATE EVENT delete_retirees
ON SCHEDULE EVERY 30 SECOND
DO
BEGIN
    DELETE
    FROM employee_demographics
    WHERE age >= 60;
END $$

DELIMITER ;
```

---

### 📌 Check Results

```sql id="te9"
SELECT *
FROM employee_demographics;
```

👉 Rows with age ≥ 60 will be deleted automatically every 30 seconds

---

## ⚙️ Event Scheduler Check

```sql id="te10"
-- Check if event scheduler is ON or OFF
SHOW VARIABLES LIKE 'event%';
```

👉 If it shows OFF, enable it:

```sql id="te11"
SET GLOBAL event_scheduler = ON;
```

---

## ⚠️ Important Notes

* Triggers run **automatically** when data changes
* Events run **automatically based on time**
* `NEW` keyword refers to newly inserted data in trigger
* Event scheduler must be ON for events to work

## 🔧 Common Issue

* If DELETE or UPDATE does not work:

  * Uncheck: **Safe Updates**
  * Apply and reconnect
