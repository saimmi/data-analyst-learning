## 🔹 CASE for Categorization

```sql id="cs2"
-- Categorize employees based on age
SELECT 
    first_name, 
    last_name,
    age,
    CASE 
        WHEN age <= 30 THEN 'Young'
        WHEN age BETWEEN 31 AND 50 THEN 'Adult'
        WHEN age >= 50 THEN 'Senior'
    END AS age_bracket
FROM employee_demographics;
```

👉 `CASE` works like IF-ELSE conditions

## 🔹 CASE for Salary Calculation

### 📌 Rules:

* Salary < 50000 → 5% increase
* Salary ≥ 50000 → 7% increase
* Finance department → 10% bonus

---

```sql id="cs3"
SELECT 
    first_name, 
    last_name, 
    salary,

    -- Calculate new salary after increment
    CASE
        WHEN salary < 50000 THEN salary * 1.05
        WHEN salary >= 50000 THEN salary * 1.07
    END AS new_salary,

    -- Bonus for Finance department (dept_id = 6)
    CASE 
        WHEN dept_id = 6 THEN salary * 0.10
    END AS bonus

FROM employee_salary;
```

## 🧠 Key Notes

* `CASE` is used for conditional logic
* Works like IF → ELSE IF → ELSE
* `END` is mandatory
* Can be used in `SELECT`, `WHERE`, `ORDER BY`
* Returns value based on condition

---

* `CASE` → apply conditions and return custom output
